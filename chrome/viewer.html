MyLogger 帮助

            
本工具用于本地日志查看、过滤、定位、标记、保存和断点日志筛查。

          

          

            
          

        

        

          

            
打开日志

            
点击右上角“打开文本/日志”，或将日志文件拖拽到主窗口。支持 `.log`、`.txt`、`.json`、`.md` 等常见文本文件。

          

          

            
文件与分析菜单

            
顶部“文件”菜单支持打开文本/日志和打开文件夹。打开多个文件时会生成多个标签页；顶部过滤、搜索、筛选、保存等操作只影响当前可见标签页。“分析”菜单提供日志比较入口。

          

          

            
过滤与搜索

            
顶部“过滤”会直接影响主窗口展示结果，可输入关键字，也可勾选“正则”和“区分大小写”。“搜索”会基于当前过滤结果打开搜索结果子窗口。

          

          

            
时间筛选

            
点击表头“时间”打开时间筛选窗口。时间点按当前日志实际分布生成，选择起点和终点后点击“确定”，主窗口只展示对应时间段内的日志。

          

          

            
级别筛选

            
点击表头“级别”打开级别选择器，可多选日志级别。选择后会立即过滤主窗口日志。

          

          

            
Tag 筛选

            
点击表头“Tag”打开 Tag 筛选窗口。输入 Tag 后点击“确定”才会生效。点击“+”可增加多个 Tag，多个 Tag 按“任意一个完全匹配”进行过滤。

          

          

            
查看上下文

            
当主窗口存在过滤条件时，单击某一行日志会打开上下文子窗口，展示该日志前后各 50 行，并高亮当前行。

          

          

            
标记与跳转

            
点击行号可标记或取消标记当前行。右上角“全部标记”可查看标记行。搜索区旁边的上下箭头可在标记行之间跳转。

          

          

            
复制与保存

            
双击日志行会复制当前行文本。右上角“保存过滤结果”会把当前主窗口展示的过滤结果保存为 `.txt` 文件。各子窗口右上角“保存”也默认保存为 `.txt`。

          

          

            
日志比较

            
在“分析”菜单点击“比较”，选择两个不同的已打开日志后进入左右分屏比较窗口。左右两侧支持独立过滤、正则、大小写、行号标记、标记上下跳转和 Jump 行号跳转；双击日志行可复制当行文本。

          

          

            
常用过滤与配置

            
左侧“常用过滤”可保存、编辑、删除、拖拽排序、导入和导出过滤规则，点击条目确认后会覆盖顶部过滤输入框。右上角设置可统一管理服务地址、语言、常用过滤、历史过滤，并支持备份和恢复配置。

          

          

            
后台分析

            
左侧“后台分析”默认收起，点击可展开。选择断点 JSON 文件后，可点击“获取断点日志”从本地后台服务提取断点日志字符串。

          

          

            
断点日志筛查

            
顶部“过滤”支持多行输入，每一行都是一条过滤规则，多行之间按“任意一条命中”过滤主窗口日志。勾选“正则”时，每一行会作为独立正则参与匹配。过滤输入框右侧“查看”用于打开左右分屏明细窗口。

          

          

            
本地服务

            
后台分析服务地址默认为 `http://127.0.0.1:7878/analyze`。服务状态绿灯表示本地服务可用，红灯表示不可用。点击“获取断点日志”时，插件会将断点 JSON 内容传给后台；后台根据 JSON 中的断点信息对比本地源码，找到断点对应源码行中的日志字符串并返回，并自动填入顶部“过滤”刷新主窗口。

          

          

            
实用工具

            
展开左侧“后台分析”后，点击“实用工具”可打开随插件打包的本地工具页。该页面包含 JSON 格式化、参数处理、二维码生成等常用小工具，可在不离开插件环境的情况下辅助处理日志和调试文本。
