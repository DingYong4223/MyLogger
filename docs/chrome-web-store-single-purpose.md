# MyLogger Chrome Web Store 单一用途说明

## 可直接填写版本

MyLogger 的单一用途是帮助开发者在本地打开、查看、过滤、搜索和分析日志文件，以便快速定位日志中的关键信息。

插件的所有功能都围绕这一用途展开：用户可以打开本地日志文件，按关键字、正则、时间、级别和 Tag 过滤日志，查看搜索结果及上下文，标记和跳转重要日志行，保存筛选结果，并管理常用过滤规则。可选的断点日志功能仅用于从用户选择的断点 JSON 中提取日志过滤规则，并将这些规则用于当前日志筛查。

MyLogger 不用于修改浏览器设置，不改变网页内容，不提供与日志查看和日志分析无关的功能。

## English Version

MyLogger has a single purpose: helping developers open, view, filter, search, and analyze local log files so they can quickly identify relevant log information.

All features directly support this purpose. Users can open local log files, filter logs by keyword, regular expression, time range, level, and tag, inspect search results and surrounding context, mark and jump to important log lines, save filtered results, and manage reusable filter rules. The optional breakpoint log feature is only used to extract log filter rules from a user-selected breakpoint JSON file and apply those rules to the current log analysis workflow.

MyLogger does not modify browser settings, does not alter web page content, and does not provide functionality unrelated to log viewing and log analysis.

## 更短版本

MyLogger 的单一用途是帮助开发者在本地查看、过滤、搜索和分析日志文件。插件提供的文件打开、日志过滤、搜索、上下文查看、标记、保存、常用过滤和断点日志规则提取功能，均直接服务于日志分析这一用途。

## 填写建议

Chrome Web Store 的 Privacy 页会要求声明扩展的 single purpose。根据 Chrome Web Store 发布说明，Privacy 页用于声明扩展的单一用途和用户数据处理方式；官方质量指南也要求扩展的用途应狭窄、清晰、易理解。

建议优先使用上面的“可直接填写版本”。如果输入框长度有限，可以使用“更短版本”。
